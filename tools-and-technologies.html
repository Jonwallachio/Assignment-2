Tools and Technologies

Software Tools

  • GNU Emacs
    • Callum has at least a year of experience with editing text in Emacs, and it is his text editor of choice. In this project, Callum uses Emacs for writing and running/testing python code, as well as writing HTML.
  • Visual Studio Code
    • VSCode is Alejandro’s text editor of choice, which he uses for writing and running/testing python code. He did not have any experience with VSCode prior to starting these courses, however it was very easy to learn and intuitively designed. The fact that Microsoft owns both VSC and GitHub makes for a smooth experience between writing code, debugging, launching virtual environments and version control with GitHub. VSC also has a lot of plugins available that make things easier or more intuitive on the writing and testing side of things.
  • Tkinter
    • Tkinter is an interface allowing Python to utilise the Tk python library, and was used in the pygui project. Alejandro had no previous experience creating GUIs hence it’s a pretty small program, however learning Python in the programming course made for a much smoother transition to the wrapper.

    Online Tools & Services

  • GitHub
    • The team uses GitHub for file sharing, team work, and the creation of this website. None of the team has had prior experience with Github.
  • Microsoft Teams
    • The team uses MS Teams for meetings and group messages concerning the assignments. None of the team has had prior experience with Teams.
